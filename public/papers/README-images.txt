Place diagram images here using paths from QP JSON (shown under each question in the app).

Folder per paper session, e.g.:
  public/papers/wbi11-may-2019/q1.png
  public/papers/wbi11-may-2019/q1-1a.png
  public/papers/wbi11-may-2019/q1-1b_i.png

Run: npm run enrich:qp-images
to refresh image paths on all *-qp.json files.
